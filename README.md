# OpenAgeProof

An open protocol for privacy-preserving age assurance.

Age verification laws are rolling out globally. The systems being built to enforce them store government IDs and biometric data in centralized databases — exactly the kind of infrastructure that gets breached. OpenAgeProof is an alternative: age assurance anchored to credit card possession, with cryptographic guarantees that no party learns more than they need to.
